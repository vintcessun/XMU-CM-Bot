# XMU-CM-Bot

## 目的

本bot是为了服务所有转专业的同学，减少查找文件的难度和方便进群信息确认使用的。

## 功能

目前已有的功能

- Login 登录
- Logout 退登
- Download 下载文件
- Search 搜索文件（开发中...）
